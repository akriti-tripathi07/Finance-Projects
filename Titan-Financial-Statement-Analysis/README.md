# Titan Financial Statement Analysis

## Project Overview

This project presents a comprehensive Standalone Financial Statement analysis of Titan Company Limited using its published Annual Reports. The analysis evaluates the company's financial performance, profitability, liquidity, solvency, efficiency, and cash flow trends over multiple financial years.

## Objectives

- Analyze Titan's financial position and operating performance.
- Evaluate key financial ratios and year-on-year trends.
- Assess the company's liquidity, profitability, solvency, and efficiency.
- Interpret financial statements to support investment and business decisions.

## Tools Used

- Microsoft Excel
- Financial Statement Analysis
- Ratio Analysis
- Trend Analysis
- Annual Report Data

## Key Analysis Performed

- Balance Sheet Analysis
- Statement of Profit & Loss Analysis
- Cash Flow Statement Analysis
- Liquidity Ratios

- Trend Analysis
- Financial Interpretation and Findings
